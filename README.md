# VEX V5 High Strength Shaft Drilling Jig 

**Full credit to pronoob024 for the initial file and for the idea**

I would highly recommend checking out his video here: https://youtu.be/TEUXXG1_aYY


Instructions:
3D print the jig in PLA and first drill using a 3.5mm drill bit into the axle
Then, use a 4mm drill to widen the hole you have created
You can now use the VEX screws to attach the shaft to other parts


Edits I have made to Daniel's file:
• Changed the sizes of the holes to 3.5mm and 4mm
• Used text to mark the 3.5mm and 4mm holes
